Segundo apartado

  1. Significa que un elemento es hijo directo de otro, por ejemplo en este mismo codigo el elemento de lista es hijo directo de la lista ordenada, todo lo que se haga en el css se vera en el elemento li si se escribe ol > li
  2. Significa que se seleccionan elementos que se encuentran al mismo nivel, que son hermanos. Por ejemplo el h1 y el div "contenedor" del primer apartado, las modificaciones del css se veran en el h1 si se escribe .contenedor + h1
